# Sound Portfolio

2022-2 캡스톤 프로젝트 사운드기반 포트폴리오 플랫폼

<br>

## 🎵 설치 필요 프로그램

Node.js

<br>

## 🎵 실행 방법

### CLIENT

`cd cient`<br>
`npm i`<br>
`npm run start`<br>
<br>

### SERVER

`server/.env` 파일 생성

`cd server`<br>
`npm i`<br>
`npm run start`<br>

<br>
